# PickSniff

Fragrance matchmaking web app - [picksniff.com](https://picksniff.com).
Free quizzes that match beginners to fragrances from a hand-curated 750-bottle library.

- **CLAUDE.md** - operational rules and commands (start here)
- **PROJECT.md** - architecture and design decisions
- **GAPS.md** - known-issues audit with statuses
- **Design.md / BrandGuidelines.md** - design system
- **docs/** - gameplan, audits, security, legal, catalog pipeline

```bash
npm run dev      # dev server
npm test         # match-engine tests
npx next build   # production build (required before every push; push = deploy)
```
